# mill-interceptor Documentation

`mill-interceptor` lets Mill impersonate common build-tool entrypoints such as Maven, Gradle, and sbt.

This site covers user-facing guidance for:

- installing and running the interceptor
- configuring Maven plugin forwarding into Mill
- understanding release artifacts and launcher behavior
